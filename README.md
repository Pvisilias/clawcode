# ClawCode — Desktop Workspace for OpenClaw

**ClawCode is the desktop workspace built on top of OpenClaw.**
It gives OpenClaw users a full multi-project dashboard, role-based agents, planning tools, task management, and one-click git actions — all in one place.

## What is ClawCode?

OpenClaw is a powerful AI coding agent runtime. But when you're running multiple projects — several apps, clients, or side projects — managing OpenClaw across all of them gets messy fast.

ClawCode wraps OpenClaw in a desktop app designed for that exact workflow:

- Switch between projects instantly from a sidebar
- Each project gets its own isolated OpenClaw agent setup, auto-generated on connect
- One central orchestration chat that can coordinate agents across all projects
- Built-in Plan mode, Workflow builder, and Task list — all editable by you or your agents
- Agent loops that run queued tasks automatically until completion
- Running agents view, logs, recent changes, and cross-project git activity in one dashboard
- Stage, commit, and push without leaving the app
- Built-in terminal, quick-open to VS Code, and predefined automations

**ClawCode is not an editor.** It is a multi-project operations layer on top of OpenClaw.

## How it works

```
OpenClaw (required, free)
    ↓
ClawCode (desktop app)
    ├── Project A — Planner agent, Coder agent, UI agent, Bug fixer agent
    ├── Project B — Planner agent, Coder agent, UI agent, Bug fixer agent
    ├── Project C — ...
    └── Central orchestration agent (accesses all projects)
```

1. Install OpenClaw and configure your model provider (GPT-4o, Claude, Gemini, etc.)
2. Install ClawCode
3. Connect your project folders — ClawCode generates agent configs automatically
4. Plan work, queue tasks, run agents, ship from the dashboard

## Key features

### Multi-project dashboard
One sidebar. All projects. No terminal switching, no folder jumping.

### Project-isolated OpenClaw agents
Every project gets its own agent set. Agents can't bleed context between projects.

### Role-based agents per project
- **Planner** — breaks goals into plans, workflows, and tasks
- **Coder** — writes and modifies code based on task instructions
- **UI Agent** — handles frontend, layout, and design work
- **Bug Fixer** — diagnoses and resolves errors and regressions

### Central orchestration
One orchestration chat that can instruct any project agent. Queue work across projects, monitor everything from one view.

### Agent loops
Queue a set of tasks and let agents run until completion — no babysitting required.

### Built-in planning system
- **Plan mode** — define outcomes, direction, milestones
- **Workflow builder** — design repeatable step-by-step processes
- **Task list** — convert plans into executable tasks, assign to agents

### In-dashboard git
Stage, commit, push. No terminal needed. Recent changes view shows diffs per project.

### Predefined automations
Set up repeatable workflows once — run them anytime from any project.
